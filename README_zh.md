## Description
每次，我学习一种技术，都要搭建相应的开发环境。其他人也在重复着我的步骤。所以我想将我搭建环境的步骤尽量记录下来，后来者可以一键搭建开发环境，这样可以节省大量时间，将精力集中到学习新东西上来。如上。同时这里也当作我的代码仓库。
